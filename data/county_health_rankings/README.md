# county_health_rankings

This is a dcf data source project, initialized with `dcf::dcf_add_source`.

Standard files are pulled from the [PopHIVE/county_health_rankings](https://github.com/PopHIVE/county_health_rankings) repository. Run `ingest.R` to download or update them.

You can use the `dcf` package to check the project:

```R
dcf_check()
```

And process it:

```R
dcf_process()
```
